# 💰 Salary Predictor (TND)

A sleek, AI-powered salary prediction application built with **React** and **Python Flask**. The app estimates  salaries in **Tunisian Dinar (TND)** based on experience, skills, location, and professional level.

![Salary Predictor Screenshot](https://github.com/user-attachments/assets/4d42fff5-ef30-461d-abbe-a30f05dd1bc6)

---

## ✨ Features

- **🎯 Accurate Predictions** — Machine learning model trained on Tunisian tech market data
- **🌙 Dark Glassmorphism UI** — Modern aesthetic with animated money particles and gradient effects
- **⚡ Real-time Calculation** — Instant salary estimates based on your profile
- **📍 Location Aware** — Supports Tunis, Sfax, Sousse, Nabeul, and Remote positions
- **🛠️ Skills Analysis** — Factors in technologies like Python, React, Docker, MongoDB, etc.
- **📱 Fully Responsive** — Optimized for desktop and mobile devices
